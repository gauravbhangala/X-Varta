## 💡 Pro Tips

1. **Colors**: All in `tailwind.config.js`
2. **Fonts**: All in `globals.css` (line 1)
3. **Content**: All in component files
4. **Animations**: All in `globals.css`
5. **Responsive**: Use Tailwind breakpoints (sm:, md:, lg:)
6. **Performance**: Never use `!important`, use layers
7. **Accessibility**: Use semantic HTML
8. **TypeScript**: Use strict mode for safety
